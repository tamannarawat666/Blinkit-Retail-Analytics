# 🛒 Blinkit Retail Analytics Dashboard

An end-to-end **Retail Data Analytics Project** built using **SQL, Python (Pandas & NumPy), and Power BI**. The project analyzes Blinkit's retail sales data to uncover business insights, monitor key performance indicators (KPIs), and build an interactive dashboard for decision-making.

---

## 📌 Project Overview

This project demonstrates the complete data analytics workflow—from data cleaning and SQL analysis to Python-based exploration and Power BI dashboard development. The dashboard helps analyze sales performance, outlet efficiency, product categories, and customer ratings.

---

## 🛠️ Tech Stack

- **SQL (MySQL)** – Data Cleaning & Business Analysis
- **Python** – Pandas, NumPy
- **Power BI** – Dashboard Development & DAX

# 💡 Business Insights

- Supermarket Type 1 generated the highest sales.
- Tier 3 outlets contributed the highest revenue.
- Fruits & Vegetables and Snack Foods were the best-performing product categories.
- Medium-sized outlets achieved the highest sales contribution.
- Interactive slicers allow users to analyze sales across multiple business dimensions.
